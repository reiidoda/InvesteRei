# Support

## Getting Help
- Read project docs in [`docs/`](docs/README.md).
- Use repository issues for bug reports and feature requests.
- For security matters, follow [`SECURITY.md`](SECURITY.md) and avoid public disclosure.

## Issue Reporting Checklist
- Environment details (OS, runtime versions, Docker version).
- Steps to reproduce.
- Expected vs actual behavior.
- Relevant logs or screenshots.
